# Code of Conduct

The Helmdeck project follows the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**. The full text is canonical and is incorporated by reference into this project.

## Scope

This Code applies in all project spaces — issue trackers, pull requests, chat, mailing lists, events — and when an individual is officially representing the project in public spaces.

## Reporting

Concerns about a contributor's conduct can be raised by emailing the maintainer **Tosin Akinosho** at **tosin.akinosho@gmail.com** with the subject prefix `[helmdeck-coc]`. Reports will be reviewed and investigated promptly and fairly. Reporters' identities will be kept confidential.

For security vulnerabilities, see [`SECURITY.md`](SECURITY.md) instead.

## Enforcement

Maintainers will follow the [Community Impact Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) published with the Contributor Covenant when determining the consequences for any action they deem in violation of this Code.

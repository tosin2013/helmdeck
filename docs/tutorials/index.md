# Tutorials

Learning-oriented walkthroughs. Read these first if you're new to helmdeck — each one takes you from zero to a working setup with explicit steps and expected outputs.

## Available tutorials

- **[Pack demo playbook](../integrations/pack-demo-playbook.md)** — 20 copy-pasteable prompts that exercise every shipped capability pack against a live install. Use it after the install script to verify your environment.

## What goes here

Tutorials are *learning-oriented*. They are the most prescriptive of the four documentation modes:

- They have a clear start and end.
- The reader follows the steps without making decisions.
- The goal is competence-by-doing, not exhaustive coverage.

If you want a recipe for a specific problem, see **[How-to guides](../howto/)**. If you want to look up a pack contract, see **[Reference](../reference/)**. If you want to understand *why* helmdeck is shaped the way it is, see **[Explanation](../explanation/)**.
